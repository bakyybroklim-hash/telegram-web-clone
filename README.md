# Telegram Web Clone Ultra

Premium React/Vite messenger interface inspired by Telegram Web.

## Included

- Three-panel desktop layout: navigation, chat list, conversation and inspector.
- Mobile layout with fullscreen conversation and slide-in chat list.
- Light and dark themes.
- Search, sorting, unread counters, online status, pinned and muted states.
- Working message composer with Enter-to-send.
- Message reactions, pinned message strip and drag-and-drop file feedback.
- Settings, profile, files, AI, calls, security and admin-ready UI surfaces.
- Browser notification permission action.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Verify

```bash
npm run lint
npm run build
```

The current version builds successfully with Vite.
